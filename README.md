## Overview

### What is Reactive Network?

Reactive Network is an event-driven blockchain platform that fundamentally changes how smart contracts execute:

- **Traditional EVM**: Transaction-driven execution (smart contracts are passive programs)
- **Reactive Network**: Event-driven execution (smart contracts are reactive agents)

### Key Differences

| Dimension | EVM | Reactive |
|-----------|-----|----------|
| Trigger | Transaction | Event / State Change |
| Execution | Active call | Auto-trigger |
| Contract Role | Passive program | Reactive agent |
| Automation | Depends on bots | Native automation |

## Key Takeaways

> **Reactive + AI = Autonomous On-chain Agents**

The combination enables:

This architecture creates the foundation for **Autonomous On-chain Economy**.
